## セットアップ

### 必要な環境変数

`.env` ファイルを作成してください（`.env.example` を参考）：

```bash
cp .env.example .env
```

編集が必要な項目：
- `ANTHROPIC_API_KEY` — Anthropic API Key
- `OPENAI_API_KEY` — OpenAI API Key
- `GITHUB_PERSONAL_ACCESS_TOKEN` — GitHub Personal Access Token（デプロイ用）

### インストール

```bash
pip install -r requirements.txt
```

### 開発サーバー実行

```bash
# Backend (FastAPI)
uvicorn app.main:app --reload

# Frontend (Streamlit)
streamlit run frontend/app.py
```

### テスト実行

```bash
python -m pytest tests/ -v
```

## MCP サーバ設定

VS Code ユーザー設定にMCPサーバが設定済み：
- **GitHub MCP** — リポジトリ操作
- **File System MCP** — ファイルシステム操作
- **Git MCP** — Git コマンド実行
- **HTTP/REST MCP** — Web API 呼び出し

設定ファイル: `~/.vscode/settings.json`

## セキュリティ

✅ `.env` は `.gitignore` で除外（ローカル保管）
✅ `.env.example` はプレースホルダーのみ（公開安全）
✅ API キーは環境変数で管理
